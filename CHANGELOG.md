# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-05-13

### Added

- **naming-conventions** — Universal naming rules across all surfaces
- **cli-creator** — Node.js CLI scaffolding with commander, config, tests
- **tdd** — Test-driven development with red-green-refactor loop
- **crystallize** — Turn deep work into reusable skills and patterns
- **domain-model** — Domain modeling stress-testing with terminology sharpening
- **grill-me** — Relentless plan/design interview until shared understanding
- **enterprise-artifact-generator** — Governed enterprise artifacts (MD, PPTX, PDF, relay packets)
- **intelligence-report** — Structured intelligence briefs with Mermaid, tables, scenarios
- **pptx-agent** — PowerPoint creation, editing, and validation
- **design-an-interface** — Multi-design generation via parallel sub-agents
- **shadcn** — shadcn/ui init, components, registries, theming
- **cloudflare-worker-builder** — Cloudflare Workers scaffolding with Hono, Vite, bindings
- **github-triage** — Issue triage via label-based state machine
- **setup-pre-commit** — Husky + lint-staged pre-commit hooks
- **caveman** — Ultra-compressed communication (~75% token reduction)
- **disk-guard** — Disk space emergency response and prevention
- **here-now** — Instant static web hosting with proxy routes
- **algorithm-state-engine** — Interactive algorithm visualizations (canvas, timeline, zero deps)
- **skill-surgery-rd** — Skill audit, repair, merge, and gap analysis
- **ubiquitous-language** — DDD glossary extraction with ambiguity flags
- Per-skill guides in `guides/` directory

## [1.0.0] - 2026-05-13

### Added

- **gh-recent** — GitHub recent activity resolver with multi-account support and skill creation pipeline
- **write-a-skill** — Skill authoring toolkit with frontmatter, progressive disclosure, and validation
- **browser-surface-bridge** — 4-layer authenticated browser data access, Tailscale-gated
- **remotion-render** — High-fidelity visual rendering from React/HTML to PNG/PDF
- **langgraph-agent** — Production-grade stateful AI agent builder with LangGraph + LangChain
- Plugin marketplace support (`.claude-plugin/marketplace.json`)
- Postinstall script (`scripts/install.js`)
- Initial README and CHANGELOG
