# pptx-agent — Guide

Create, edit, and validate PowerPoint presentations programmatically.

## Methods

- **python-pptx** — programmatic PPTX creation
- **html2pptx** — HTML to PPTX conversion
- **Template workflows** — branded template fill

## Usage

```
Create a PPTX for [topic]
Build a deck from [data]
Validate this PPTX
```
