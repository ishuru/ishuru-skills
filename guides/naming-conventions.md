# naming-conventions — Guide

Auto-applies naming rules across all surfaces: sessions, chats, files, branches, commits, slides, email subjects.

## Convention Categories

- **Sessions**: `YYMMDD-kebab-case-purpose`
- **Files**: lowercase kebab-case, no spaces
- **Branches**: `type/scope` (feat/, fix/, chore/, docs/)
- **Commits**: conventional commits (feat:, fix:, chore:)
- **Slides**: Sentence case, action-oriented

## Usage

Triggers automatically on any naming action. No explicit invocation needed.
