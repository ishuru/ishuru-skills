# crystallize — Guide

Turn completed deep work into reusable skills, patterns, or documentation.

## Trigger

- "crystallize", "capture this as a skill", "make this reusable"
- After any session where the agent solved something non-trivial

## Workflow

1. Identify the reusable core
2. Extract the pattern from the session
3. Write as SKILL.md or pattern doc
4. Validate with skill_validate
